Plan de implementación — pliego M5 «tablas»

2026-07-06 · Previos: M0-M4 ✅ (main, CI verde, 390 tests; documento objetivo completo) · Ejecución: subagent-driven · Objetivo: HTML ajeno/email-style (el PDF Tubuencamino original es maquetación de tabla) y el sustrato de Bootstrap (M6+).

Goal: tablas CSS 2.2 §17 usables de verdad: table/thead/tbody/tr/td/th, algoritmo de anchos AUTO (min/max-content vía IntrinsicSizer — ya existe) y FIXED, colspan, padding/bordes/fondos por celda, vertical-align top/middle/bottom, border-spacing (modelo separado), UA defaults (th bold+center), y paginación partiendo ENTRE filas (filas atómicas). Más el housekeeping diferido de M4.
Exclusiones M5 con warning (nunca silencio): border-collapse: collapse (M6 — separado es el default CSS), rowspan (M6), repetición de thead por página (M6), caption/colgroup/col, células anónimas complejas (§17.2.1 completo — M5 hace la variante mínima: texto suelto en tr/table se envuelve en celda/fila anónima).

  • Todo lo de M0-M4 vigente (strict_types, VOs readonly, px→pt solo Pdf\, deptrac, sin datos personales, trailer, TDD 4-gates, baselines tests-only, código = spec anotada §17.x).

M5-T4 — TableFormattingContext: anchos y layout de celdas

CSS 2.2 §17.5.2. Nº columnas = max de Σcolspan por fila. AUTO: por celda min/max-content (IntrinsicSizer sobre sus children + paddings/borders de celda); colspan reparte su min/max entre sus columnas (proporcional al max de las simples, o partes iguales si 0 — documentar); ancho de tabla = width declarado o min(cap del containing, Σmax); distribución: si Σmax ≤ disponible → cada col su max (+ sobrante proporcional SOLO si width declarado mayor, per §17.5.2.2); si Σmin ≥ disponible → cada col su min (overflow permitido); intermedio → min + proporcional a (max−min). FIXED: primera fila manda: width de celda o partes iguales del resto; sin IntrinsicSizer (rápido). Celdas: cada celda se layoutea con BlockFlowContext (lazy, patrón M4) con containing width = Σanchos de sus columnas + spacing interno de colspan − paddings/borders de celda (width-override para imponerlo); altura de fila = max(alturas de celda); las celdas se ESTIRAN a la altura de fila (geometry-only, patrón stretch M4, vertical-align llega en T5 — por ahora top). border-spacing horizontal entre columnas y vertical entre filas + alrededor (modelo separado §17.6.1). Fondos/bordes de celda pintan (BoxFragment normal); fondo de fila detrás de celdas; fondo de tabla detrás de todo. Tests hand-computed: 2 cols auto contenido asimétrico; width declarado reparte; overflow por Σmin; fixed partes iguales; colspan 2; spacing 6px. Commit: feat(layout): table formatting context with auto and fixed column algorithms
